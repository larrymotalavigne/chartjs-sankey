Chart.js Sankey Diagram Examples

1. Energy Flow (hover + tooltips)

Hover over any flow to see a tooltip. Non-hovered flows dim automatically. Hover over a node to highlight all connected flows and dim the rest.

2. Budget Allocation (gradient + nodeColors)

Uses colorMode: 'gradient' with a nodeColors map. Flow colors transition from source node color to destination node color. Labels show values via labels.formatter.

3. Customer Journey (click events + styled nodes)

Click on flows or nodes. Uses nodeColor callback, nodeBorderRadius, colorMode: 'to', and flowLabels.
